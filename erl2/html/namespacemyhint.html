Check_consistency.

Callback for the service server /checkconsistency.

Parameters
reqboolean
Returns
True: if any hypothesis is consistent, False otherwise

If any id has 3 or more hints, those are uploaded to the ontology and checked for completeness and incosistency. The ID of consitent hypotesis is returned. If at least one ID has returned a consistent hypothesis the response is "true"
